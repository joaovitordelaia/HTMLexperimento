Usando Imagens

Para usar imagens, se usa a tag 'img', não tem uma closing tag.

Ela tem 2 atributos obrigatórios.

  • src: que especifico o caminho da imagem
  • alt: que especifica um texto alternativo a imagem

Usando Links

Usando a tag 'a' , é possivel criar um elemento que ao ser clicado
leva o usuario a outra pagina que o dev desejar.

Um link pode ser uma imagem ou um texto, pois a tag 'a' cria um hiperlink desses elementos.
É importante usar o atributo href para indicar o link do link


Absolute URLs vs. Relative URLs

  • Absolute URLs

    URL completa com protocolo (http, https) e domínio

  • Relative URLs

    Caminho relativo à página atual ou ao domínio

Comportamento ao clicar

  • _self: abre na mesma pagina!

  • _blank: aparecem em uma nova aba

  • _parent: Opens the document in the parent frame

  • _top: Opens the document in the full body of the window
